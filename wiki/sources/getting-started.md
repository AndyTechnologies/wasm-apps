# Fuente: Tutorial de primeros pasos

**Ruta**: `docs/tutorial/getting-started.md`
**Propósito**: Tutorial para crear la primera app nativa WebAssembly desde cero.

## Contenido extraído

- Prerrequisitos: Node.js ≥ 22, pnpm, CMake + Ninja, toolchain C++
- 6 pasos: crear directorio → wapp init → escribir `.wasm.ts` → wapp setup → wapp build → ejecutar
- Ejemplo completo con función factorial y console.log
- Siguientes pasos: configurar proyecto, WASI, compilación cruzada

Ver [[overview|Overview]] para la síntesis del proyecto.
